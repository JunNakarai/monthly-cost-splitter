# Monthly Cost Splitter

月別の折半費用をローカルブラウザ内で計算・保存する小規模Webアプリです。

## Features

- 月別の費用入力
- localStorage 保存
- 月別グラフ
- 計算内訳のコピー
- CSVエクスポート/インポート
- GitHub Pages 向け static export

## Development

```bash
npm install
npm run dev
```

## Privacy

入力データはブラウザの localStorage に保存されます。公開リポジトリには個人の金額データを含めない構成です。
